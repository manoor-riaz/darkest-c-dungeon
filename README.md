# Darkest C Dungeon

## Description
Développement d’une version simplifiée du jeu *Darkest Dungeon* en langage C, dans le cadre d’un projet académique.

Le jeu consiste à gérer une équipe de personnages affrontant des ennemis lors de combats au tour par tour, avec gestion des statistiques, du stress et des équipements.

## Fonctionnalités
- Système de combat au tour par tour
- Gestion de plusieurs personnages avec classes et statistiques
- Gestion des points de vie et du stress
- Système d’équipements (accessoires)
- Progression avec nouveaux personnages et ennemis
- Interface en mode terminal

## Technologies
- C
- Structures de données
- Listes chaînées

## Contexte du projet
Projet réalisé en binôme dans le cadre de la Licence Informatique.

Implémentation basée sur un cahier des charges imposant l’utilisation de structures et de listes chaînées pour gérer les personnages, les combats et les équipements.

## Résultat
Projet académique validé avec une note de 14/20.

## Compétences développées
- Manipulation des pointeurs et structures en C
- Implémentation de listes chaînées
- Gestion de mémoire
- Conception d’un système de jeu structuré
- Organisation d’un projet en langage bas niveau
